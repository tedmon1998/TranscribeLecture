# Требования к ресурсам для методов распознавания речи

## Обзор методов

Программа поддерживает несколько методов распознавания речи, каждый из которых имеет разные требования к ресурсам.

---

## 1. Whisper (OpenAI)

### Общие характеристики
- **Тип**: Локальная обработка (офлайн)
- **Поддержка языков**: 99+ языков, включая русский
- **Точность**: Очень высокая
- **Интернет**: Не требуется (после загрузки модели)

### Модели Whisper

#### tiny (39M параметров)
- **RAM**: ~1 GB
- **VRAM (GPU)**: ~1 GB
- **CPU**: Низкая нагрузка
- **Скорость**: Очень быстро (~10x реального времени на CPU)
- **Точность**: Базовая
- **Рекомендации**: Для быстрого тестирования, низкие требования

#### base (74M параметров)
- **RAM**: ~1.5 GB
- **VRAM (GPU)**: ~1.5 GB
- **CPU**: Средняя нагрузка
- **Скорость**: Быстро (~5x реального времени на CPU, ~30x на GPU)
- **Точность**: Хорошая
- **Рекомендации**: Баланс скорости и качества (рекомендуется)

#### small (244M параметров)
- **RAM**: ~2 GB
- **VRAM (GPU)**: ~2 GB
- **CPU**: Высокая нагрузка
- **Скорость**: Средняя (~2x реального времени на CPU, ~15x на GPU)
- **Точность**: Очень хорошая
- **Рекомендации**: Для лучшего качества при ограниченных ресурсах

#### medium (769M параметров)
- **RAM**: ~5 GB
- **VRAM (GPU)**: ~5 GB
- **CPU**: Очень высокая нагрузка (медленно)
- **Скорость**: Медленная (~0.5x реального времени на CPU, ~5x на GPU)
- **Точность**: Отличная
- **Рекомендации**: Для максимального качества, требуется GPU

#### large (1550M параметров)
- **RAM**: ~10 GB
- **VRAM (GPU)**: ~10 GB
- **CPU**: Экстремальная нагрузка (очень медленно)
- **Скорость**: Очень медленная (~0.2x реального времени на CPU, ~2x на GPU)
- **Точность**: Превосходная
- **Рекомендации**: Для профессионального использования, обязательно нужен GPU

### GPU ускорение

#### Apple Silicon (M1/M2/M3) - MPS
- **Поддержка**: Автоматическая
- **Ускорение**: ~5-10x быстрее CPU
- **Ограничения**: Word timestamps отключены (техническое ограничение MPS)
- **Рекомендации**: Отлично работает с моделями base, small, medium

#### NVIDIA GPU - CUDA
- **Поддержка**: Требуется PyTorch с CUDA
- **Ускорение**: ~10-30x быстрее CPU
- **Ограничения**: Нет
- **Рекомендации**: Лучший вариант для больших моделей (medium, large)

#### CPU
- **Скорость**: Базовая
- **Рекомендации**: Подходит для tiny и base моделей

## 2. Системный распознаватель - Google Speech Recognition

### Характеристики
- **Тип**: Облачная обработка (онлайн)
- **Поддержка языков**: 100+ языков, включая русский
- **Точность**: Очень высокая
- **Интернет**: Обязательно требуется

### Требования к ресурсам
- **RAM**: ~100 MB (только для клиента)
- **VRAM (GPU)**: Не требуется
- **CPU**: Минимальная нагрузка
- **Скорость**: Очень быстро (~1-2 секунды на 5 секунд аудио)
- **Интернет**: Стабильное соединение (минимум 1 Mbps)
- **Лимиты**: Бесплатно до ~50 запросов/день (затем платно)

### Преимущества
- ✅ Не требует GPU
- ✅ Не требует много RAM
- ✅ Очень быстро
- ✅ Высокая точность
- ✅ Поддержка многих языков

### Недостатки
- ❌ Требует интернет
- ❌ Данные отправляются в Google
- ❌ Лимиты на бесплатное использование

### Рекомендации
- Для быстрой записи с микрофона
- Когда есть стабильный интернет
- Для экономии ресурсов компьютера

---

## 3. Системный распознаватель - PocketSphinx

### Характеристики
- **Тип**: Локальная обработка (офлайн)
- **Поддержка языков**: Только английский (из коробки)
- **Точность**: Средняя
- **Интернет**: Не требуется

### Требования к ресурсам
- **RAM**: ~200 MB
- **VRAM (GPU)**: Не требуется
- **CPU**: Низкая нагрузка
- **Скорость**: Очень быстро (~0.5-1 секунда на 5 секунд аудио)
- **Размер моделей**: ~50-100 MB (для английского)

### Преимущества
- ✅ Работает офлайн
- ✅ Не требует GPU
- ✅ Минимальные требования к RAM
- ✅ Очень быстро
- ✅ Приватность (все локально)

### Недостатки
- ❌ Только английский язык
- ❌ Средняя точность
- ❌ Русский язык не поддерживается

### Рекомендации
- Только для английского языка
- Для офлайн режима
- Для устройств с ограниченными ресурсами

## 4. Системный распознаватель - macOS Speech Recognition

### Характеристики
- **Тип**: Системная обработка (офлайн)

- **Интернет**: Не требуется (после загрузки языковых пакетов)

### Требования к ресурсам

- **VRAM (GPU)**: Не требуется (использует системные ресурсы)
- **CPU**: Низкая-средняя нагрузка
- **Скорость**: Быстро (~1-2 секунды на 5 секунд аудио)
- **Платформа**: Только macOS

### Преимущества
- ✅ Работает офлайн
- ✅ Не требует GPU
- ✅ Низкие требования к RAM
- ✅ Быстро
- ✅ Поддержка русского языка
- ✅ Приватность (все локально)

### Недостатки
- ❌ Только macOS
- ❌ Требует загрузки языковых пакетов

### Рекомендации
- Для macOS пользователей
- Для офлайн режима с русским языком
- Когда нужна приватность

## Рекомендации по выбору метода

### Для записи с микрофона (русский язык)

#### Офлайн режим (без интернета)
1. **Whisper base** - лучший баланс (рекомендуется)
   - Требует: 1.5 GB RAM, GPU опционально
   - Скорость: Быстро
   - Точность: Хорошая

2. **Whisper small** - для лучшего качества
   - Требует: 2 GB RAM, GPU рекомендуется
   - Скорость: Средняя
   - Точность: Очень хорошая

#### Онлайн режим (с интернетом)
1. **Google Speech Recognition** - самый быстрый
   - Требует: 100 MB RAM, интернет
   - Скорость: Очень быстро
   - Точность: Очень высокая

### Для записи ВКС/системного звука

#### Офлайн режим
- **Whisper base** или **small** (в зависимости от доступных ресурсов)

#### Онлайн режим
- **Google Speech Recognition** (быстрее и экономит ресурсы)

### Для устройств с ограниченными ресурсами

#### macOS
- **macOS Speech Recognition** (если доступен)
- **Whisper tiny** (если macOS Speech недоступен)

#### Другие платформы
- **Google Speech Recognition** (если есть интернет)
- **Whisper tiny** (если нет интернета)

### Для максимального качества

- **Whisper large** + NVIDIA GPU
- Требует: 10 GB RAM, 10 GB VRAM, мощный GPU
- Точность: Превосходная

## Примеры использования ресурсов

### Сценарий 1: MacBook Air M1 (8 GB RAM)
- **Рекомендуется**: Whisper base с MPS
- **Альтернатива**: Google Speech Recognition (если есть интернет)
- **Не рекомендуется**: Whisper medium/large (недостаточно RAM)

### Сценарий 2: MacBook Pro M2 (16 GB RAM)
- **Рекомендуется**: Whisper small с MPS
- **Альтернатива**: Whisper medium с MPS (для лучшего качества)
- **Также подходит**: Google Speech Recognition

### Сценарий 3: ПК с NVIDIA GPU (8 GB VRAM)
- **Рекомендуется**: Whisper small/medium с CUDA
- **Альтернатива**: Whisper large с CUDA (для максимального качества)

### Сценарий 4: Слабый компьютер (4 GB RAM, без GPU)
- **Рекомендуется**: Google Speech Recognition (если есть интернет)
- **Альтернатива**: Whisper tiny (если нет интернета)

## Оптимизация производительности

### Для Whisper
1. Используйте GPU (MPS для Apple Silicon, CUDA для NVIDIA)
2. Выберите подходящую модель (не больше, чем нужно)
3. Используйте более короткие чанки для системного распознавателя (5 сек)
4. Используйте более длинные чанки для Whisper (30 сек) для лучшей точности

### Для системного распознавателя
1. Используйте короткие чанки (5 секунд) для быстрой записи
2. Перекрытие между чанками предотвращает потерю слов
3. Дедупликация удаляет повторяющиеся слова

## Заключение

Выбор метода зависит от:
- **Доступных ресурсов** (RAM, GPU)
- **Наличия интернета**
- **Требований к точности**
- **Требований к скорости**
- **Требований к приватности**

Для большинства случаев рекомендуется:
- **Whisper base** для офлайн режима
- **Google Speech Recognition** для онлайн режима
