# Быстрый старт веб-приложения

## 1. Установка зависимостей

### Python зависимости:
```bash
pip install -r requirements.txt
```

### Node.js зависимости:
```bash
cd frontend
npm install
```

## 2. Запуск

### Быстрый запуск (рекомендуется):
```bash
./start.sh
```

Этот скрипт автоматически:
- Проверит зависимости
- Установит их при необходимости
- Запустит сервер и фронтенд
- Покажет ссылки для доступа

Нажмите `Ctrl+C` для остановки.

### Ручной запуск:

**Терминал 1 - Бэкенд сервер:**
```bash
python3 server.py
```
Сервер запустится на `http://localhost:8000`

**Терминал 2 - Фронтенд:**
```bash
cd frontend
npm run dev
```
Фронтенд запустится на `http://localhost:5173` (или другом порту)

## 3. Использование

1. Откройте браузер: `http://localhost:5173`
2. Выберите настройки (источник, метод, язык)
3. Нажмите "▶ Начать запись"
4. Текст появится в реальном времени!

## Примечания

- Для системного звука на macOS нужен BlackHole
- Первый запуск Whisper займет время (загрузка модели)
- Google распознавание требует интернет
